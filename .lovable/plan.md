# Time of L&L · 情侣专属电子手账

一个浪漫、富有仪式感的双人手账网页，主打"纪念日实时计数 + 拍立得照片墙 + 旅行时光轴"三大核心体验。

## 视觉方向（待你确认或调整）

- **氛围**：奶油米色背景 + 蜜桃粉/玫瑰金主色 + 深酒红点缀，柔和、温暖、复古手账风
- **字体**：标题用衬线 `Cormorant Garamond` / `Instrument Serif`（手写感强），正文用 `Plus Jakarta Sans`，点缀手写体 `Caveat` 用于"亲笔签名"式标注
- **质感**：纸张纹理背景、淡淡的格子线、手撕胶带、邮票、爱心贴纸、蝴蝶结等装饰元素
- **动效**：进入时拍立得"掉落"动画、悬停时轻微倾斜、翻转查看背面、滚动视差、心跳呼吸效果

> 如果你想看 2–3 个不同风格的草稿（例如：复古手账风 / 极简法式 / 水彩日系），告诉我，我会先生成可视化方案再开工。

## 页面结构

```text
/                  首页 — 纪念日计时器 + 拍立得照片墙轮播
/timeline          旅行记录 — 垂直时光轴
/album   (可选)    完整拍立得相册网格（可后续添加）
```

### 首页 `/`
- **顶部 Hero**：
  - 大字标题 "Time of L&L"
  - **纪念日实时计时器**：天 / 时 / 分 / 秒，带心跳脉冲动画
  - 副标题如 "Since YYYY-MM-DD · 我们一起走过的每一刻"
- **拍立得照片墙**：
  - 自动套拍立得白边相框（底部留白手写日期/地点）
  - 轮播 + 散落式排布，轻微随机旋转
  - 点击放大；再次点击 / 悬停翻转，背面显示心情文字
- **小亮点**：飘落花瓣或星光粒子、底部签名档

### 旅行记录 `/timeline`
- 垂直时光轴，中间一条带爱心节点的浪漫主线
- 每个节点卡片：日期 · 地点 · 主图（拍立得风格）· 故事文字 · 心情 tag
- 滚动时节点淡入 + 主线绘制动画
- 支持左右交错布局，桌面端浪漫感更强

## 技术细节

- TanStack Start + Tailwind v4，新增路由 `src/routes/index.tsx`、`src/routes/timeline.tsx`
- 在 `src/styles.css` 中扩展奶油/玫瑰金/酒红语义 token 与纸张纹理背景
- 组件：`AnniversaryCounter`、`PolaroidCard`（含翻转）、`PolaroidWall`（轮播 + 散落）、`Timeline`、`TimelineNode`
- 计时器用 `useEffect` + `setInterval(1000)`，SSR 安全（mount 后启动）
- 照片占位：用 `imagegen` 生成符合情侣旅行氛围的示意照片，放在 `src/assets/`
- 动画：Tailwind 自带 keyframes + 少量 `framer-motion`（用于翻转 / 入场）
